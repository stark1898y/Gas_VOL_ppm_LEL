计算公式

1 %VOL = 10000 ppm
%LEL = %VOL / LEL(%VOL) x 100

示例（甲烷，LEL = 5.0 %VOL）：
  10%LEL = 5.0% x 10% = 0.5%VOL = 5000 ppm
  100%LEL = 5.0%VOL = 50000 ppm
